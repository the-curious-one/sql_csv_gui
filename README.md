## Wecome

Do you hate the inability to run SQL in Excel/Google Sheets, but like the standard worksheet UI?

Build your data in Excel files/Google sheets, export as CSV, and filter it with through it with SQL here! 

I already know how to use SQL reasonably well, so this project is very simple for now, but the goal is to make SQL queries user friendly.
